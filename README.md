# `vue-h5-template`

一个基于 Vue 的 H5 应用模板。

## 安装方式

- 克隆代码

```shell
git clone https://github.com/l246804/vue-h5-template.git [可选，项目名]
```

- 安装依赖

```shell
pnpm i
```

- 升级依赖并清除无效文件

```shell
pnpm run plop # 执行后选择 app-generator
```

- 生成 lint 工具和其他配置项

```shell
pnpm run plop # 执行后选择 config-generator
```
